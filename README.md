# Peppy (Working Title)

Peppy is beyond just a normal mobile app, Peppy is a non-ai, non-invasive, companion app, designedto help you stop your bad habits!

## Goals
- Detect when a user opens blacklisted apps (Android MVP).
- Interrupt with configurable friction (delay, challenge, reflection).
- Track usage locally and show daily summaries.
- Privacy-first: no cloud sync in MVP.

## Non-goals (MVP)
- iOS app monitoring
- Social features
- Cloud accounts
- Ads or monetization

## Tech Stack
- Kotlin
- Jetpack Compose
- Android SDK
- Room (local storage)
